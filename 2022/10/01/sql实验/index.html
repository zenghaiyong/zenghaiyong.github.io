    1.使用企业管理器或sql语句创建一个名为“SM”的数据库，初始大小为10MB，最大为50MB，数据库自动增长，增长方式按10%；

    create database SM1

    实验二：创建表

    注：identity(1,1) 自动增长 1

    1. 在数据库SM中创建学生表student,课程表course,选课表sc
      student(sid,sno,clno,sname,ssex,sage,sbir)
      说明：sid int identity(1,1) 序号
      sno 为主关系键，为字符类型 学号
      clno 字符类型， 班级号
      sname 字符类型，并不为空
      ssex 字符类型，check的值的范围为男女
      sbir 日期类型 出生日期
      sage int；
      sql语句：

      use SM1
create table student
( sid int identity(1,1),
  sno varchar(6) PRIMARY KEY,
  clno varchar(10),
  sname varchar(12) not null,
  ssex  char(2) constraint xbgl check(ssex='男' or ssex='女'),
  sbir date,
  sage int
  )

    2. course(cno,cname,ccredits,ctno,cpno,ctime)
      说明：cno 字符类型，主关系键
      cname 字符类型，唯一键
      ccredits 学分，精确数值型，精确长度为2，小数位为0
      ctno（课程类型） ，cpno 字符类型
      ctime 日期类型

      sql语句：

      use SM1
create table course
( cno varchar(6) PRIMARY KEY,
  ccredits  decimal(2,0) constraint xl UNIQUE,
  ctno  char(10),
  cpno  char(10),
  ctime date
  )

    3. sc(sno,cno,score)
      说明：sno+cno为主键，并且sno是student的外部键，cno是course的外部键。
      score精确数值型，精确长度为3，小数位为1

      sql语句：

    use SM1
create table sc
( sno varchar(6) constraint sc_sno_for foreign key references student(sno),
  cno  varchar(6) constraint sc_cno_for foreign key references course(cno),
  score decimal(3,1),
  constraint sc_sno_cno_pk primary key(sno,cno)
 
  )

    查询s表中学号倒数第5个数字为9，最后一个数在1至5之间的学号

    select sno from student
where sno like'%9____' and sno like'%[0-9]'

    revoke CREATE TABLE
on database sm
from yy

    创建表tbl_operator及其表tbl_operator_old(做备份用)

    use SM1
create table tbl_operator

  )

  )

    该表的表包含列：姓名，身份证号码，注册日期，最后登录日期，各列均不为空
    本表为一个网站人员注册表
    要求：
    1、创建一个存储过程对tbl_operator进行维护，把指定日期内未登录的人员信息转移到
    表tbl_operator_old中并删除原始数据；
    2、该存储过程需要带参数。

    use sm1
go
create proc weihu5
 ( @jiange int)
as
select *
from tbl_operator  
where  DATEDIFF(DAY,lasttime,GETDATE())>@jiange

insert into tbl_operator_old 
select *
from tbl_operator  
where  DATEDIFF(DAY,lasttime,GETDATE())>@jiange

delete 
from tbl_operator  
where(name in(select name
from tbl_operator  
where  DATEDIFF(DAY,lasttime,GETDATE())>@jiange))

exec weihu5 10

    实验十一触发器实验

    说明：

    例: 在系别表（department表）中增加一个学生人数字段(snumber)表示该系当前最新的学生人数，该字段的值随着学生信息表中的记录数发生改变，即：当学生表(student表)中新增学生记录，并且分配了具体的所属系别后，该系的学生人数自动加1；当学生表中删除某记录并且删除的记录原有所属系别时，该系的学生人数自动减1；当学生信息表中的所属系别值发生改变时，原来系别的学生人数自动减1，新的系别的学生人数自动加1。
    以上处理要求分别用insert、delete、update触发器实现其处理功能。
    student表: Sno（学号）， Sn（姓名）， Sex（性别），Age （年龄）， Depart（系编号）。
    department表：Deno（系编号）， Dname（系名）， Lead（系主任）， Tel。

    建表：

    [age] [int] ,
    [Depart] [nvarchar] (10) )

CREATE TABLE [department] (
    [Deno] [nvarchar] (10)  NOT NULL ,

    [Lead] [nvarchar] (10) ,

    触发器

    create  trigger de_num
on student
for insert,update,delete
as 
begin
 update department 
 set snumber=snumber-1 where deno in (select depart from deleted)
 update department 
 set snumber=snumber+1 where deno in (select depart from inserted)
 end
